# create-awesome-api
用于快速创建api模块的脚手架

## 快速开始

#### with yarn

```bash
yarn create awesome-api my_api
# or
# npx create-awesome-api my_api -p npm
cd my_api
yarn start
```

#### with npm

```bash
yarn create awesome-api my_api -p npm
# or
# npx create-awesome-api my_api -p npm
cd my_api
npm run start
```

## 构建

```bash
# in project directory
yarn build
```
